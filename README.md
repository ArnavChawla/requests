### INSTALLATION
#### REQUIREMENTS
- [Python version 3.7+]()
- [Pip]() (Should be intalled along with Python)

```
git clone https://www.github.com/appcypher/feature-requests.git
```

```
cd feature-requests
```

```
pip3 install pipenv
```

```
pipenv install
```

```
pipenv shell
```

```
flask run
```

### RUNNING DOCKER FILES
#### REQUIREMENTS
- [Docker]()

```
docker-compose -f docker/compose.yml up
```
